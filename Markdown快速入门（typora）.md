## 二、标题

```go
//# 一级标题
//## 二级标题
//### 三级标题
//#### 四级标题
//##### 五级标题
//###### 六级标题

//最后一个“#” 之后要留一个空格
```

## 三、字体

```C++
// 加粗
**[加粗的内容]**
//代码高亮显示--注意：这里要自己在【文件】--》【偏好设置】--》【Markdown】 中勾选上【高亮】
==[高亮的内容]==
//删除线
~~[删除的内容]~~
//斜体
*[斜体的内容]*
```

// 加粗
**[加粗的内容]**
//代码高亮显示

==高亮的内容==

//删除线
~~[删除的内容]~~

*[斜体的内容]*
